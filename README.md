# lsw-filesystem

Filesystem tool based on IDB for LSW.

## Installation

```sh
npm i -s @allnulled/lsw-filesystem
```

## Importation

From node.js:

```js
require("@allnulled/lsw-filesystem");
```

From html:

```html
<script src="node_modules/@allnulled/lsw-filesystem/lsw-filesystem.js"></script>
```
